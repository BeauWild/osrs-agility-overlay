# OSRS Agility Overlay v0.1.0

Initial public release.

## Download

Download `OSRSAgilityOverlay-win-x64.zip`, extract it, then run `OSRSAgilityOverlay.exe`.

## First-time setup summary

Set up RuneLite first:

- Visual Ticks installed and moved into a corner.
- Visual Ticks circles placed on the same spot using negative horizontal spacing.
- Blindfold enabled.
- Entity Hider enabled.
- Agility plugin turned off, or clickboxes/Mark of Grace highlighting disabled.
- Tile Indicators off.
- Screen Markers off if used.
- Minimap minimised.
- Camera locked/positioned so the full Ardougne course can be clicked without moving camera.

Then run the overlay, press F10, map or adjust markers from the end of each movement, set each tick delay, set world lag, then use Set tick pixel for Visual Ticks sync.

## Notes

This is a visual overlay only. It does not click, move the mouse, press keys, automate gameplay, read game memory, modify RuneLite, or modify the OSRS client.

# Changelog

## v0.1.0

Initial public GitHub release.

Based on the final v60 overlay info layout build.

### Included

- Default Ardougne Agility Course markers.
- F10 marker editor.
- Tick-based marker countdowns.
- Manual marker delay adjustment.
- World lag and queue safety adjustment.
- Optional single-pixel Visual Ticks sync.
- Minimal/info overlay toggle.
- Lap stats, combo stats, best time, and missed time display.
- GitHub Actions build/release workflow.
- One-click local build scripts.
